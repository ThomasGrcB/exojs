Exojs

Welcome to the JavaScript Exercises series designed to help you grasp the core concepts of JavaScript programming. Whether you're just starting out or looking to reinforce your skills, these exercises will guide you through various levels of complexity, gradually expanding your understanding of JavaScript.

This series covers a wide range of topics, from basic data types to more advanced concepts like loops, functions, objects, and callback functions. Each exercise is carefully crafted to provide you with hands-on experience and insights into the fascinating world of JavaScript.

Remember that learning programming is a journey, and the best way to learn is by doing. For each exercise, we encourage you to create a separate folder named "exercise[exercise number]" (e.g., "exercise1", "exercise2", etc.). Inside each folder, place the relevant HTML and JavaScript files. Don't hesitate to experiment, make mistakes, and discover the solutions on your own.

Each exercise comes with tasks to complete, and you're encouraged to tackle them to the best of your abilities. Feel free to modify the exercises, experiment with different approaches, and seek further challenges beyond what's provided here. Happy coding!

Exercise 8: Code Analysis Task

Title: Object Manipulation

Description: Given an object and code snippet, manipulate the object's properties as per the code's instructions. 

            
                // Object
                let person = {
                    name: "Alice",
                    age: 30,
                    profession: "Engineer"
                };
    
                // Code Snippet
                function updatePerson(obj) {
                    obj.age += 5;
                    obj.profession = "Senior " + obj.profession;
                }
    
                updatePerson(person);
                console.log(person);

Tasks:

  1. Explain what the `updatePerson` function does.
  2. Predict the output of the `console.log` statement.
  3. Explain the changes made to the `person` object after calling `updatePerson`.
